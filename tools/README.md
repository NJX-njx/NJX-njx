# Banner Tooling

The current GitHub profile banner is generated by `generate_crt_banner.py`.

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
```

## Generate

```powershell
.\.venv\Scripts\python tools\generate_crt_banner.py
```

The script reads `images/portraits/portrait-reference.jpg` and writes
`assets/banners/crt-banner.gif`.
